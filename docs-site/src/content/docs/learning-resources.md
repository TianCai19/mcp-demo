# 学习资源

这里整理一些高质量的 MCP 学习资源，方便快速上手和深入理解。

## FastMCP 快速开始

如果你想用 FastMCP 构建自己的 MCP Server，推荐先阅读官方 Quickstart：

- [FastMCP Quickstart](https://gofastmcp.com/v2/getting-started/quickstart)

**建议阅读顺序**：
1. 通读 Quickstart，了解最小可运行示例
2. 回到本项目的 `weather/weather.py` 对照实现
3. 尝试加一个新工具（例如“空气质量查询”）
